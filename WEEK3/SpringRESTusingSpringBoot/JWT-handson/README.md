# 🔐 JWT Authentication - Authentication Service

## 📖 Overview

This project demonstrates the first step of JWT authentication using Spring Boot and Spring Security. A secured authentication endpoint is created using HTTP Basic Authentication. Valid users can access the `/authenticate` endpoint successfully.

---

## 🎯 Objective

- Configure Spring Security.
- Create an authentication endpoint.
- Protect REST APIs using HTTP Basic Authentication.
- Verify authentication using Browser and Postman.

---

## 🛠️ Technologies Used

- Java 21
- Spring Boot
- Spring Security
- Spring Web
- Maven
- IntelliJ IDEA
- Postman

---

## 🌐 REST Endpoint

**Method:** GET

```
http://localhost:8090/authenticate
```

---

## 📚 Concepts Covered

- Spring Security
- HTTP Basic Authentication
- SecurityFilterChain
- Authentication Controller
- Protected REST Endpoint

---

## ✅ Result

Successfully implemented a secured authentication endpoint using Spring Security. The endpoint was verified using Postman with Basic Authentication credentials.
