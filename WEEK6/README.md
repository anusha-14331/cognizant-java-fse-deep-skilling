# Hands-on 6 – Services & Dependency Injection

## Topics Covered

- Angular Services
- Dependency Injection (DI)
- Singleton Services
- Hierarchical Dependency Injection
- Shared Data Store
- Component-Level Providers

## Features

- Created CourseService for managing course data
- Created Course model interface
- Displayed live course count on Home page
- Implemented Course Summary Widget
- Created EnrollmentService for enroll/unenroll functionality
- Displayed enrolled courses in Student Profile
- Implemented NotificationService using component-level providers
- Demonstrated Singleton and Hierarchical Dependency Injection

## Technologies Used

- Angular 20
- TypeScript
- HTML
- CSS

## Learning Outcome

Learned how Angular Services share data across components, how Dependency Injection works, how singleton services are created using `providedIn: 'root'`, and how component-level providers create separate service instances.
