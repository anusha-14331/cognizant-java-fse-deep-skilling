# 📘 Angular Hands-on 3 – Directives & Pipes

## 📌 Objective

Implement Angular structural directives, attribute directives, custom directives, and custom pipes in the Student Course Portal.

---

## Topics Covered

### Structural Directives

- *ngIf
- *ngFor
- *ngSwitch
- ng-template

### Attribute Directives

- ngClass
- ngStyle

### Custom Directive

- HostListener
- Highlight Directive

### Pipes

- Built-in Pipes
- Custom Pipe
- PipeTransform

---

## Features Implemented

### Task 1 – Structural Directives

- Loading screen using *ngIf
- Displayed courses using *ngFor
- Performance optimization using trackBy
- Status badges using *ngSwitch
- No courses template using ng-template

### Task 2 – Attribute Directives

- Dynamic classes using ngClass
- Dynamic border colors using ngStyle
- Expand/Collapse course cards
- Getter method for cleaner templates

### Task 3 – Custom Directive & Pipe

- Highlight directive with configurable hover color
- Credit Label custom pipe
- Displayed:
  - 1 Credit
  - Multiple Credits
  - No Credits
