# 📘 Angular Hands-on 2 – Data Binding, Lifecycle Hooks & Component Communication

## 📌 Objective

Implement Angular data binding techniques, lifecycle hooks, and parent-child communication.

---

## Topics Covered

- String Interpolation
- Property Binding
- Event Binding
- Two-Way Binding (ngModel)
- ngOnInit
- ngOnDestroy
- ngOnChanges
- @Input
- @Output
- EventEmitter

---

## Features Implemented

### Task 1 – Data Binding

- Displayed portal title using interpolation
- Enabled/Disabled button using property binding
- Displayed enrollment message using event binding
- Implemented course search using ngModel

### Task 2 – Lifecycle Hooks

- Used ngOnInit to initialize component
- Used ngOnDestroy to detect component removal
- Used ngOnChanges to detect input changes

### Task 3 – Parent-Child Communication

- Passed course data using @Input
- Sent events using @Output
- Used EventEmitter for enrollment events
- Displayed selected course ID
